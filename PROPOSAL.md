# 地震學第一天課程提案：AI 增強學習環境

## 課程概覽
本提案概述了一種創新的地震學入門教學方法，採用現代 AI 工具和程式設計實踐。本課程將傳統地震學概念與尖端技術相結合，創造出引人入勝、實用的學習體驗。

## 學習目標

### 第一天目標
1. **環境設置**：學生將配置其地震學資料分析的開發環境
2. **AI 輔助學習**：學習運用 AI 工具（Gemini、NotebookLM）進行知識萃取
3. **程式設計基礎**：地震學的 Python 程式設計入門
4. **資料探索**：基礎地震學資料處理和視覺化

## 創新教學方法

### 1. Google Colab 整合（「反重力」平台）
我們將使用 Google Colab 作為主要教學平台：
- **無需安裝**：學生可以立即開始編程
- **雲端運算**：存取 GPU 進行資料處理
- **內建 AI 助理**：Gemini 整合提供即時幫助
- **協作功能**：共享和協同處理筆記本

### 2. AI 代理整合
學生將學習使用 AI 作為學習夥伴：

#### Gemini AI 助理
- **程式碼生成**：學習提示 AI 生成地震學專用程式碼
- **概念解釋**：詢問地震學原理問題
- **除錯協助**：獲得程式碼錯誤的協助
- **文件搜尋**：快速查找函式庫功能

#### NotebookLM 知識萃取
- **教科書處理**：上傳地震學教科書以獲得 AI 驅動的摘要
- **問答環節**：詢問有關特定章節或概念的問題
- **學習指南**：生成個人化學習材料
- **概念映射**：理解地震學概念之間的關係

### 3. 實作學習路徑

#### 階段一：環境設置（30 分鐘）
```
1. Access Google Colab
2. Connect to runtime
3. Install seismological libraries
4. Test installation with sample code
```

#### 階段二：AI 輔助學習（45 分鐘）
```
1. Setup NotebookLM with course materials
2. Practice querying AI for seismology concepts
3. Extract key information from textbooks
4. Generate study summaries
```

#### 階段三：基礎地震學編程（60 分鐘）
```
1. Load seismic data files
2. Visualize waveforms
3. Calculate basic parameters (amplitude, frequency)
4. Plot results
```

#### 階段四：互動練習（45 分鐘）
```
1. Earthquake location exercise
2. Magnitude calculation practice
3. Wave propagation simulation
4. Group discussion and Q&A
```

## 課程計劃範例

### 分鐘級進度表

**0-15 分鐘：歡迎與設置**
- 地震學和課程目標介紹
- 存取 Google Colab
- 測試基本 Python 指令
- AI 工具概覽

**15-30 分鐘：AI 工具介紹**
- 使用課程教科書設置 NotebookLM
- 練習向 AI 提問
- 學習有效的提示技巧
- 從教科書章節中萃取關鍵概念

**30-45 分鐘：地震學的 Python 基礎**
- 變數和資料類型
- 列表和陣列
- 基礎繪圖
- 檔案 I/O 操作

**45-90 分鐘：第一個地震學程式**
- 使用 ObsPy 載入地震資料
- 理解波形資料結構
- 建立時間序列圖
- 計算基本統計

**90-120 分鐘：引導式練習**
- 進行地震分析練習
- 遇到困難時使用 AI 協助
- 與全班分享結果
- 討論發現

**120-150 分鐘：互動環節**
- 學生展示（選用）
- 在 AI 協助下進行問答
- 預覽下一堂課
- 作業分配
